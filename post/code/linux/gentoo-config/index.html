“masked by amd64”

方法如下：

1
2
3
4
5
6
7

cd /etc/portage/package.keywords
sudo touch dev-util  <== 文件名是你被屏蔽程序所在的包的名字，例如：dev-util/nemiver 对应dev-util,
sudo echo "dev-util/nemiver ~amd64" > dev-util

/etc/portage/package.unmask 
#添加关闭mask对这个版本的屏蔽，并启用这个软件包
=sys-devel/gcc-4.9.0-r2
