Urxvt复制粘贴

2023-09-15
linux配置
约 227 字 预计阅读 1 分钟 次阅读

文章目录

关于复制粘贴操作有两种方法：

  1. 通过快捷键：复制 alt-ctrl-c，粘贴 atl-ctrl-v，
  2. 图形界面X11支持一种独特的复制粘贴方式，即如果你在另一个程序比如文本编辑器中使用鼠标拖动来高亮一段文字后，不用进行任何操作，此时选中的内容已经复制到剪贴板中了，随后在rxvt中单击鼠标中键即可将选中内容粘贴到其中，如果鼠标没有中键，可以同时按下左右键以达到同样的效果，还可以使用Shift+Insert组合键来完成粘贴；反之，从rxvt中向外复制内容同样如此。
