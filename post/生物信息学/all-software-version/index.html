R SessionInfo()

Ok, this one’s a bit ridiculous, but it works. It’s the Unix chainsaw in action. Essentially this uses a terrific tool called Ag to search my book’s code for anytime I’ve called library() to load a package, turn this into a clean set of library calls for all packages in the book, use this to load in these packages, and then print the session info. This uses process substitution, which I teach in Chapter 7.
