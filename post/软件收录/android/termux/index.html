什么是 Termux？

Termux 是一款运行于 Android 系统的开源终端模拟器。它提供了一个 Linux 环境，即使设备不具备 root 权限也能使用。通过自带的包管理器（pkg、apt），Termux 可以安装许多现代化的开发和系统维护工具，例如 zsh、Python、Ruby、NodeJS、MySQL 等软件。

Termux 的名字来自于 “终端模拟器”（terminal emulator）和 “Linux 终端”（Linux terminal）的缩写。它最初由一位名叫 Fredrik Fornwall 的瑞典开发者开发，并在 2015 年 11 月发布了第一个版本。

特点：

  • 开源：Termux 是开源软件，其源代码在 GitHub 上公开，任何人都可以自由使用、修改和分享。
  • 无需 root：Termux 可以无需 root 权限运行，这使得它更安全、更可靠。
  • 丰富的软件包：Termux 自带的包管理器可以安装各种软件包，满足用户的不同需求。

如何使用
